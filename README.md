3 Maven Archetypes to create java app, web app, and java library.

Java app features :
- Shade plugin.
- Executable jar.
- log4j2.

Web app features :
- Tomcat7 plugin.
- Spring MVC.
- log4j2.
- test url : http://localhost:8080/hello

Java Library :
- log4j2.

Install using : mvn install<br/>
To create project, run : mvn archetype:generate -DarchetypeCatalog=local
